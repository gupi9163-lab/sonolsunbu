# 📱 Tətbiqi Quraşdırma Təlimatı

## ✅ YENİ: Quraşdırma Düyməsi Həmişə Görünür!

Artıq sayta daxil olanda **yaşıl quraşdırma bölməsi** həmişə görünəcək!

---

## 📲 Necə Quraşdırım?

### 🤖 Android (Chrome)

1. **https://bdu-calculator.pages.dev** açın
2. Saytın yuxarısında **yaşıl quraşdırma bölməsini** görəcəksiniz
3. **"Quraşdırma Təlimatları"** düyməsinə basın
4. Avtomatik quraşdırma başlayacaq

**və ya Manuel:**
1. Yuxarı sağ küncdə **3 nöqtəyə (⋮)** basın
2. **"Ana ekrana əlavə et"** seçin
3. **"Quraşdır"** düyməsinə basın
4. Ana ekranda ikon görünəcək

---

### 🍎 iOS (Safari)

1. **Safari brauzerdə** https://bdu-calculator.pages.dev açın
2. Saytın yuxarısında **yaşıl quraşdırma bölməsini** görəcəksiniz
3. **"Quraşdırma Təlimatları"** düyməsinə basın
4. Təlimatları oxuyun

**Manuel Quraşdırma:**
1. Aşağıda ortada **"Paylaş" düyməsinə (⬆️)** basın
2. **"Ana Ekrana Əlavə Et"** seçin
3. **"Əlavə et"** düyməsinə basın
4. Ana ekranda ikon görünəcək

---

### 💻 Desktop (Chrome, Edge, Brave)

1. **https://bdu-calculator.pages.dev** açın
2. Ünvan sətrinin sağında **quraşdırma ikonu (⊕)** görünəcək
3. İkona basın və **"Quraşdır"** seçin

**və ya:**
1. Sağ yuxarı küncdə **3 nöqtəyə** basın
2. **"BDU Hesablama Platformasını quraşdır"** seçin

## ❓ Tez-tez Verilən Suallar

### Quraşdırma düyməsi görünmür?
- ✅ Artıq həmişə görünür! Yaşıl bölmə saytın yuxarısındadır
- Əgər görmürsünüzsə, səhifəni yeniləyin (F5 və ya Refresh)

### İOS-da quraşdırma düyməsi görünmür?
- iOS-da avtomatik quraşdırma yoxdur
- Safari-də **Paylaş (⬆️)** > **Ana Ekrana Əlavə Et** işlədin

### Quraşdırdıqdan sonra silə bilərəm?
- Bəli! Adi tətbiq kimi silin:
  - **Android**: İkonu basıb saxlayın > Sil
  - **iOS**: İkonu basıb saxlayın > Tətbiqi Sil

### Yaddaş tutur?
- Çox az! Təxminən **1-2 MB**
- Oflayn istifadə üçün bəzi fayllar saxlanılır

### Yeniləmələr avtomatik olur?
- Bəli! Saytı açdıqda avtomatik yenilənir
- Yenilənmə üçün heç nə etməyə ehtiyac yoxdur
